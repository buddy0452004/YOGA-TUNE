# 🧘‍♀️ YOGA-TUNE – Smart Yoga & Fitness Tracking System

YOGA-TUNE is a yoga-themed web application designed to help users **track their fitness goals, study/work discipline, and daily progress** in a structured and motivating way.  
The project focuses on **user authentication, activity tracking, and progress monitoring** with a clean and engaging UI.

---

## 🚀 Features

- 🔐 User Registration & Login System  
- 🧘 Yoga-themed UI with smooth animations  
- 📊 Daily activity & progress tracking  
- ⏱️ Time-based goal monitoring  
- 📈 Performance feedback & reports  
- 🎨 Modern, responsive design  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python (Flask)  
- **Database:** MySQL / SQLite  
- **Tools:** Git, GitHub, XAMPP (for MySQL)
